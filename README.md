# MBE Group Website

Official MBE Group website built with React, Tailwind CSS, shadcn/ui, Framer Motion, and Base44.

## Stack
- React 18 + Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion
- React Router DOM
- Base44 (backend, functions, entities)

## Pages
- Home
- About
- Brands
- Advantages
- Contact
- Imprint / Terms / Privacy

## Languages
EN, DE, ZH, HI, VI, TR
